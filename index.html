1. Acceso al Sistema

Página de Login - Múltiples Métodos de Autenticación

El sistema ofrece diferentes formas de autenticación para mayor seguridad y comodidad del usuario.

1
2
3
4
5
Campo de Email

Función: Ingresar dirección de email registrada en el sistema.

Uso: Escribe tu email corporativo. El sistema validará el formato automáticamente.

Ejemplo: usuario@empresa.com
Campo de Contraseña

Función: Ingresar contraseña de acceso al sistema.

Uso: Escribe tu contraseña. Es sensible a mayúsculas y minúsculas.

La contraseña se muestra oculta por seguridad
Protección reCAPTCHA

Función: Verificar que el acceso es realizado por un usuario humano.

Uso: Marca la casilla "I'm not a robot". Puede requerir seleccionar imágenes.

Sistema de seguridad para prevenir acceso automatizado
Acceso con Google OAuth

Función: Autenticación rápida usando cuenta de Google.

Uso: Haz clic aquí para iniciar sesión con tu cuenta de Google corporativa.

Método rápido que no requiere recordar contraseña adicional
Recuperar Contraseña

Función: Restablecer contraseña en caso de olvido.

Uso: Haz clic aquí para recibir un enlace de recuperación en tu email.

Se enviarán instrucciones al email registrado

2. Dashboard Principal

Vista General del Sistema - Panel de Control

Pantalla principal que muestra un resumen ejecutivo de toda la operación empresarial.

1
2
3
4
Menú de Navegación Principal

Función: Acceso a todos los módulos del sistema ERP.

Uso: Haz clic en cualquier opción del menú para navegar al módulo correspondiente.

  • Productos: Gestión de inventario
  • Clientes: Base de datos de clientes
  • Pedidos: Procesamiento de ventas
  • Compras: Gestión de proveedores
Tarjeta de Productos en Stock

Función: Vista resumida del estado del inventario.

Uso: Muestra cantidad de productos activos y acceso rápido al módulo completo.

Click en la tarjeta para ir directamente al módulo de Productos
Métricas de Negocio

Función: Indicadores clave de performance (KPI's).

Uso: Monitoreo visual de ventas, inventario y rendimiento operacional.

  • Ventas del mes: Total facturado
  • Stock bajo: Alertas de inventario
  • Pedidos pendientes: Por procesar
Menú de Usuario

Función: Gestión de perfil y sesión de usuario.

Uso: Haz clic para ver opciones de perfil, configuración y cierre de sesión.

Incluye opción para cerrar sesión de forma segura

3. Gestión de Productos - CRUD Completo

Módulo de Productos - Vista de Catálogo

Interfaz principal para gestionar todo el inventario de productos del sistema.

1
2
3
4
5
Crear Nuevo Producto

Función: Agregar nuevo producto al catálogo.

Uso: Haz clic aquí para abrir el formulario de creación de productos.

Te llevará al formulario completo con todos los campos requeridos
Tarjeta de Producto

Función: Visualización de información del producto.

Uso: Muestra imagen, nombre, precio, stock y descripción del producto.

  • Nombre: Identificación principal
  • Precio: Valor de venta unitario
  • Stock: Cantidad disponible
  • Descripción: Características
Editar Producto

Función: Modificar información de producto existente.

Uso: Haz clic en el ícono de edición para modificar el producto seleccionado.

Se abrirá el formulario con los datos actuales para modificación
Eliminar Producto

Función: Remover producto del catálogo permanentemente.

Uso: Haz clic en el ícono de eliminación (aparecerá confirmación).

Esta acción no se puede deshacer - requiere confirmación
Búsqueda y Filtros

Función: Encontrar productos específicos en el catálogo.

Uso: Escribe términos de búsqueda o usa filtros para encontrar productos.

Puedes filtrar por categoría, precio, stock y más criterios

Formulario de Producto - Creación y Edición

Formulario completo para gestionar toda la información de los productos del sistema.

1
2
3
4
5
Nombre del Producto

Función: Identificador principal del producto.

Uso: Ingresa el nombre comercial del producto. Campo obligatorio.

Este campo es requerido - no puede estar vacío
Descripción del Producto

Función: Detalles y características del producto.

Uso: Describe las especificaciones, usos y beneficios del producto.

La descripción ayuda en las búsquedas y para informar a clientes
Precio de Venta

Función: Establecer el precio unitario de venta.

Uso: Ingresa el precio en formato numérico. Usa punto para decimales.

Ejemplo: 150000.00 para $150,000 pesos
Cantidad en Stock

Función: Definir inventario inicial disponible.

Uso: Ingresa el número de unidades disponibles para venta.

El sistema alertará automáticamente cuando el stock sea bajo
Guardar Producto

Función: Almacenar la información del producto en la base de datos.

Uso: Haz clic aquí después de completar todos los campos requeridos.

El producto aparecerá inmediatamente en el catálogo después de guardar

4. Módulos Adicionales del Sistema

Navegación entre Módulos

Usa el menú lateral para acceder a todas las funcionalidades del sistema ERP:

Clientes Pedidos Compras Auditoría Configuración

Módulo de Gestión de Clientes

Administración completa de la base de datos de clientes y contactos comerciales.

1
2
3
Datos Personales del Cliente

Función: Registrar información básica del cliente.

Uso: Completa nombres, apellidos, email, teléfono y datos de contacto.

  • Email: Para comunicaciones
  • Teléfono: Contacto directo
  • Fecha Nacimiento: Datos demográficos
Gestión de Direcciones

Función: Administrar múltiples direcciones por cliente.

Uso: Define direcciones fiscal, de correspondencia y de entrega.

Puedes tener diferentes direcciones para facturación y envíos
Guardar Cliente

Función: Almacenar toda la información del cliente.

Uso: Haz clic aquí para guardar el registro completo del cliente.

El cliente quedará disponible inmediatamente para ventas y pedidos

Módulo de Gestión de Pedidos

Control completo del proceso de ventas desde la creación hasta la entrega.

1
2
3
Estados de Pedidos

Función: Seguimiento visual del estado del pedido.

Uso: Monitorea el progreso: Pendiente → Enviado → Entregado → Facturado.

Cada estado tiene un color diferente para identificación rápida
Acciones Rápidas

Función: Gestionar pedidos directamente desde el listado.

Uso: Editar, cambiar estado, ver detalles o cancelar pedidos.

  • ✏️ Editar: Modificar información
  • 👁️ Ver: Detalles completos
  • 🔄 Estado: Cambiar progreso
Ver Detalles de Pedido

Función: Acceder a información completa del pedido.

Uso: Haz clic para ver productos, cantidades, precios y datos del cliente.

Desde los detalles puedes generar facturas y documentos

5. Configuración del Sistema

Panel de Configuración Global

Ajustes y parámetros generales que controlan el comportamiento de todo el sistema ERP.

1
2
3
Configuración de Inventario

Función: Parámetros de gestión de stock y alertas.

Uso: Define umbrales de stock bajo, políticas de reposición y alertas.

  • Stock Bajo: Cantidad mínima para alertas
  • Stock Mínimo: Nivel crítico de inventario
  • Alertas: Configurar notificaciones
Configuración Monetaria

Función: Ajustes de moneda y formatos numéricos.

Uso: Define moneda principal, formatos de precio y decimales.

Configura moneda local y tipos de cambio si manejas múltiples monedas
Datos de la Empresa

Función: Información corporativa para documentos.

Uso: Configura nombre, logo, dirección fiscal y datos de contacto.

Esta información aparece en facturas, cotizaciones y reportes
